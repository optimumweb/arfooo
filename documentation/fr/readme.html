Installation : moins de 5 minutes et vous êtes prêt
  1. Décompressez l'archive dans un dossier vide.
  2. Modifiez le fichier .htaccess pour activer PHP 5 sur votre serveur si nécessaire.
  3. Envoyez l’entièreté du dossier sur votre serveur avec votre logiciel FTP.
  4. Rendez-vous à l'adresse suivante : http://www.votresite.com/votredossier/install/
  5. Suivez les différentes étapes de l’installation, celles-ci vont vous permettre de créer les différentes tables de données nécessaires à votre annuaire, et également de créer votre compte admin.
  6. Une fois l’installation terminée, le script d’installation vous renverra vers la page de connexion. Connectez-vous avec l’identifiant et le mot de passe que vous avez entré durant l’installation.
  7. Renommez ou supprimez le dossier install pour des raisons de sécurité.
Mise à jour
Avant de vous lancer dans une procédure de mise à jour, assurez-vous de sauvegarder les fichiers que vous avez modifiés, ainsi que votre base de données (précaution oblige).
  1. Supprimez vos anciens fichiers Arfooo Annuaire sauf les dossiers /config/, /save/ et /uploads/
  2. Envoyez les nouveaux fichiers.
  3. Rendez-vous à l'adresse suivante : http://www.votresite.com/votredossier/install/update/ et sélectionnez le fichier correspondant à la version que vous souhaitez mettre à jour.
  4. C’est tout... sauf si
  5. Vous avez modifié votre template, des fonctions, pas de problème. Une mise à jour fichier par fichier et ligne par ligne est disponible dans le dossier documentation /fr/mise à jour/
Pré-requis
Pour faire fonctionner Arfooo Annuaire, il suffit d'avoir un serveur web (Unix/Linux/Windows/MacOS X) qui supporte :
  • PHP 5.1 et supérieur
  • MySQL 4.1 et supérieur
  • SPL (Standard PHP Librairy)
  • Librairie curl
Pré-requis en option :
  • mod_rewrite pour des urls optimisés pour le référencement.
  • GD librairie pour pouvoir générer les codes de sécrurités (captcha).
  • XML pour pouvoir générer des Flux Rss.
