Ci-dessous les différents changements survenus pour Arfooo Annuaire



version 2.0.2 (22/11/2010)

Mise à jour :
Version 2.0.2 :
1) Arfooo Annuaire est maintenant parfaitement compatible avec PHP 5.3.x
2) Ajout de la possibilité de formater la description avec du html. Sécurisé avec htmlpurifier. Merci pour cette très puissante librairie.
3) Ajout d'une fonction qui permet de vérifier la duplication de contenu. Peut être activée à la soumission. Permet également de vérifier ultérieurement si la description est dupliquée...
4) Ajout dans l'admin de la possibilité de classer un site dans un nombre illimité de catégories.
5) Ajout dans l'admin de la possibilité de passer des paramètres supplémentaires dans les URLs afin de traquer ses newsletters par exemples.
6) Ajout sur la page des résultats du moteur de recherche, dans la balise title et h1, de la requête cherchée par l'Internaute.
7) Amélioration de la pertinence des résultats du moteur de recherche
8) Correction d'un bug avec le moteur de recherche. Les configurations étaient codées en dures.
9) Ajout de la possibilité de personnaliser l'attribut alternatif des images.
10) Les images uploadées ne sont plus déformées.
11) Ajout dans les champs supplémentaire du champ URL. Options : Ancre texte et nofollow
12) Ajout dans les champs supplémentaire du champ Fichier. Permet d'uploader puis de télécharger un fichier.
13) Ajout de la possibilité de personnaliser le style des champs supplémentaire via le css.
14) Ajout de l'affichage d'étoiles pour les notes. C'est plus web 2.0 :D
15) Ajout dans l'admin de la rubrique "Gérer les sites bannis".
16) Ajout de nombreuses traductions oubliées.
17) Passage de l'API Google Maps version 2 à la version 3.
18) Amélioration de la partie newsletter.
19) Amélioration du système de template. Le nom du template est maintenant une variable.
20) Amélioration du système de session. Les sessions ne se fermeront plus du moment que l'utilisateur est en ligne.
21) Correction d'un bug qui empêchait l'admin de sélectionner un nombre illimité de mots clés.
22) Correction d'un bug dans l'admin qui empêchait de gérer l'image des sites.
23) Correction de bugs avec la soumission privilège et les modes de paiement Allopass et Paypal.
24) Correction d'un bug qui empêchait d'uploader un fichier csv pour la newsletter.
25) Correction d'un bug avec la suppression du cache et template compilé. Merci à gunner du forum!
26) Correction d'un bug avec les thumbs si l'image retournée était vide. Merci à dnd du forum!

6) Ajout dans l'admin d'un moteur de recherche pour rechercher les webmaster par leur email.
7) Correction de fautes d'orthographe.
8) Correction de divers problèmes avec les thumbs, ascreen et images.
9) Correction des problèmes d'affichage dans l'admin avec les navigateurs IE.
10) Correction de la gestion des URLs avec tynimce. De relative vers absolue.
11) Correction d'un bug avec la soumission privilège. Les webmasters sont maintenant redirigés correctement vers le formulaire de soumission après le paiement.
12) Correction d'un bug avec la soumission privilège. Lors de la sélection de l'offre, la description est maintenant chargée correctement.
13) Correction de divers problèmes avec les flux RSS.
14) Correction d'un bug qui empêchait la sélection de plus de 4 mots clés.
15) Correction d'un bug avec les referrers.
16) Correction d'un bug lors de l'édition d'un site par le webmaster. Les sous-catégories n'étaient pas chargées.
